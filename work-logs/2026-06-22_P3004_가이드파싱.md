## 구현 내용
- `parseGuidelines`: Gemini 2.0 Flash 모델을 통해 입력된 체험단 원고 가이드를 분석하여 필수/금지/해시태그 등의 항목을 JSON 형태로 반환하는 Cloud Function 신규 구현.
- `app/(influencer)/my-campaigns/[id]/page.tsx` 업데이트:
  - "가이드 요약 보기" 버튼을 클릭하면 `parseGuidelines` 함수를 호출해 파싱 결과를 반환받음.
  - 반환받은 결과(ParsedGuide)를 체크리스트 형태(체크박스)의 UI로 렌더링.
  - 각 항목별로 체크 상태를 관리하여 전체 진행률을 파악함.
  - "콘텐츠 제출하기" 버튼은 요약을 실행한 경우, **모든 체크리스트가 체크되어야만** 활성화되도록 로직 연동 완료.
