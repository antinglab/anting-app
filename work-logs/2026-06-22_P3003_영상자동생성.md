## 구현 내용
- `generateVideoScript`: Gemini 2.0 Flash를 활용하여 제품명과 캠페인 분위기를 기반으로 15초 분량의 릴스/쇼츠용 스크립트(JSON 구조)를 자동 생성하는 Cloud Function 구현.
- `generateVideo`: Creatomate API와 연동하여 사용자가 선택한 BGM 스타일과 길이에 맞게 영상 렌더링을 요청하고, Firestore `video_jobs/{id}` 문서를 생성하는 기능 구현.
- `creatomateWebhook`: Creatomate로부터 영상 렌더링 상태 이벤트를 수신받아 Firestore 문서를 업데이트하고, 성공 시 Telegram 알림 봇을 통해 다운로드 링크 전송 기능 구현.
- `/app/(influencer)/my-campaigns/[id]/create-video/page.tsx`: 인플루언서가 제품 이미지를 업로드하고 단계별로 영상 생성 및 확인, 수정(스크립트 등), 다운로드를 할 수 있는 프론트엔드 UI 페이지 구현.
