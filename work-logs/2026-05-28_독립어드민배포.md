## 구현 내용
- **독립 어드민 HTML 구현**: Next.js App Router의 라우팅 간섭 및 클라이언트 가드 처리 문제를 완벽히 해결하기 위해 Next.js 빌드 사이클과 무관하게 작동하는 순수 단일 HTML 파일 어드민을 `admin-panel/index.html`에 생성했습니다.
- **Firebase SDK 직접 연동**: CDN 방식의 Firebase v10 SDK 모듈을 가져와 Firestore에서 `crew_members` 목록을 최신순으로 읽고, 상태 필터링, 실시간 검색, 개별 상세 슬라이드오버(메모 수정/삭제 포함), KPI 카운트, CSV 내보내기 기능을 바닐라 JS 및 CSS로 완벽하게 구현하였습니다.
- **Firebase Hosting 설정 최적화**: `/admin-panel/**` 경로 요청에 대해 독립 어드민 페이지(`index.html`)가 항상 서빙되도록 `firebase.json`에 `rewrites` 설정을 적절히 지정하였습니다.
- **빌드 후 복사 자동화**: Next.js 프로젝트 빌드(`npm run build`) 후 `out` 디렉토리에 `admin-panel` 폴더를 복사하도록 조치하여 Next.js 정적 내보내기 결과와 독립 어드민 파일이 함께 배포되도록 처리하였습니다.
