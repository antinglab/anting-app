## 구현 내용
- **EmailJS 패키지 설치 및 환경 설정**: `@emailjs/browser`를 설치하고 `.env.local` 및 `next.config.js`에 EmailJS Service ID, Template ID, Public Key 설정 변수를 성공적으로 추가했습니다.
- **EmailJS 템플릿 문서화**: `/docs/emailjs-template.html`에 HTML 형식의 환영 이메일 템플릿 코드와 변수 정의 내용을 작성 및 저장했습니다.
- **텔레그램 알림 환경 설정**: `.env.local` 및 `next.config.js`에 Telegram Bot Token 및 Chat ID 변수를 추가하였으며, 설정 가이드 가이드를 `/docs/telegram-setup.md`에 자세히 문서화해 두었습니다.
- **이메일 및 텔레그램 전송 모듈 구현**: 
  - `lib/send-welcome.ts` 파일에 EmailJS SDK 연동 이메일 발송 함수 구현 (실패 시에도 가입 처리에 방해되지 않도록 예외 처리).
  - `lib/send-telegram.ts` 파일에 텔레그램 봇 API (`/sendMessage`) 호출 알림 발송 함수 구현 (실패 시 예외 처리 적용).
- **가입 폼 연동**: `components/JoinForm.tsx` 내 `handleSubmit` 함수에서 Firestore에 데이터를 정상적으로 적재한 직후, `sendWelcomeEmail`과 `sendTelegramAlert`를 비동기적으로 실행하여 가입자 대상 환영 메일 및 관리자 알림봇 메시지가 나가도록 연동을 완료하였습니다.
